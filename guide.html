Quick User Guide

What is OpenBooks?

OpenBooks is a website implemented using almost all known technologies developed for making websites. It simulates an online bookstore or library, where you can like books and check any kind of information about them. Extra simple version of Good Reads so to say.

It is divided in two main sections: blog entries and "My Library". Blog entries are pages containing general information on known books. "My Library" contains the list of all books available, your favorite ones and shops nearby.

Deployed using Microsoft Azure. Took me a while so appreciate it as much as you can, my self esteem needs it.

If you want to know how to use it, keep reading!

MY LIBRARY

Most important part of the application

All Books

Here you can find the available books of the week. Normally the collection includes new releases or editions, new additions to the top lists, etc. To view them, just click the "Show All Books" button once. The info is loaded from an XML file using AJAX and JQuery. You shouldn't have any problem while listing them. If so, my fault.

Here comes the fun part: How do I add one of these books to My Books? When you add a book to you list of favorites you will be able to get more info on it as well as buying links from other online sites. To do this, simply click on the section which contains the info of the book (all of them are listed in separate boxes), drag it to the Heart image shown at the bottom of the sidebar and drop it. The book you picked will be automatically added to your personal favorites and stored in our database, an alert will be shown. Functionality provided by the Drag&Drop HTML5 API.

My Library

Your favorite books are stored here (the ones you dragged and dropped from the list of all books). Books are listed showing their cover image. To get more info just click on one of them and a beautiful handmade dialog will appear with all the information available from the book. If any info is not present, it will be shown as "not specified".

All info on the books is obtained using Open Library's API. And the cover images as well. It is an awesome page. Check it here.

Data is processed using JQuery and display follows a flexbox style.

Shops

Want to know where you stand at the moment? Use this section. It will accurately tell you what your position is, even featuring coordinates (impressive I know), and it will show it on Google Maps (blue pin, if you click on it, it tells you the exact address).

"Show Shops", as it says, will load a list of the most visited book stores in Asturias including information such as website, phone number, description, pictures and once again coordinates. If you want to know where exactly is any of these stores located, click locate and it will immediately appear on the above map (red pin), showing address.

Shop info is loaded as HTML from an XML file but this time it is processed using PHP (XSLT, transformation using XML and XSL files).

APIs used: HTML5 Geolocation (exact location of user), Google Maps (map), Geocoder (exact location of shop).
